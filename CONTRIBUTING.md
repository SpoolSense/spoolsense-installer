# Contributing to SpoolSense Installer

Thanks for your interest in contributing! Here's how to get started.

## Reporting Bugs

Open an issue using the Bug Report template. Include the full installer output.

## Submitting Pull Requests

1. Fork the repo and create a branch from `dev`
2. Make your changes
3. Test on a Raspberry Pi — the installer interacts with hardware and system services
4. Test all install modes (Scanner + Middleware / Scanner only / Middleware only)
5. Open a PR targeting the `dev` branch (not `main`)

### Branch workflow

- `dev` — active development, all PRs target here
- `main` — production releases only, merged from dev when stable

## Questions?

Open an issue using the Question template or start a discussion.
